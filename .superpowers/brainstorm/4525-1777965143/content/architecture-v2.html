Player Identity — Two Paths

👤 Guest

  1. Enter display name
  2. Enter room code
  3. Play — scores tracked in-session only

🔑 Permanent Profile

  1. Choose username + passphrase
  2. JWT stored in browser
  3. Join any game linked to profile
  4. History + lifetime stats tracked
